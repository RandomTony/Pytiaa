Calcul de différence analogique

La différence analogique d'une composante s'obtient en effectuant l'opération :
1 - |(Ai - Bi) - (Ci - Di)|

La différence analogique finale s'obtient en faisant la somme des différence analogiques de chaque composante
DA(A, B, C, D) = sum(DA(Ai, Bi, Ci, Di)) i allant de 1 à n

Code du calcul de la différence analogique

# Analogical difference A and B
        adx1 = points[t[1]][0] - points[t[2]][0]                # A - B
        ady1 = points[t[1]][1] - points[t[2]][1]
        # Analogical difference C and D
        adx2 = points[t[3]][0] - new[0]                         # C - D
        ady2 = points[t[3]][1] - new[1]
        # Real analogical difference
        adx = 1 - abs(adx1 - adx2)                              # AD = 1 - | (A - B) - (C - D) |
        ady = 1 - abs(ady1 - ady2)
        ad = adx + ady
