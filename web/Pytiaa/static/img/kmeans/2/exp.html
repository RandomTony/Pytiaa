Calcul de distance

Le programme va calculer la distance entre notre point à classer et les différents points du dataset. Un tableau de distance est créé.

def _compute_distances(new, points):
dists = []
for i, pt in enumerate(points):
s = 0
for j, n in enumerate(pt[:-1]):
d = new[j] - n
s += d**2
dists.append([i, math.sqrt(s)])
return dists
