Création d'un dataset

Le générateur de données créer un dataset, composé de points apparetenant à plusieurs classes.

Générateur aléatoire

def random_generation( n, nbClass, dim=2):
p = []
cl = color_generation(nbClass)
for i in range(n):
p.append( [uniform(0, 1) for j in range(dim)])
p[-1].append(choice(cl))
return p
