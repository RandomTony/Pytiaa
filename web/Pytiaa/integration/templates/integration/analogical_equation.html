Qu'est qu'une équation analogique ?

Tout d'abord une analogie est une ressemblance que l'on peut faire entre deux choses

Par exemple : La tour Eiffel est à Paris ce que La Statue de la Liberté est aux Etats-Unis

Cela revient à dire : A est à B ce que C est à D, qui s'écrit de façon plus mathématique a : b :: c : d

Cas d'équations solubles (x inconnu)

a : a :: a : x solution : x = a

a : a :: b : x solution : x = b

a : b :: a : x solution : x = b

Cas d'équations non-solubles (x inconnu)

a : b :: b : x aucune solution
